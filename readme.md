Tetris
==========

### 项目介绍
	用Java的swing框架开发一个俄罗斯方块

### 项目目录结构
* config: 配置
* docs: 文档
* finally: 最后成品
* graphics: 用到的图片材料
* lib: 第三方库
* src: 源文件

### src下的包
* config: 项目配置相关类包	
* dao: 数据处理类(读数据 存数据)包		
* dto: 数据对象类包		
* entity: 实体类包
* main: 项目主入口类包
* service: 项目主逻辑类包
* ui: 项目界面类包
* ui.window: 字界面类包
* utils: 项目工具类包
